# 前端数据展示说明

## 📊 前端可以体现的数据

前端可以完整展示存储的数据，但根据用户权限和数据类型，展示的内容不同。

---

## 🎯 数据展示位置

### 1. 审核台（审核员/管理员）

**访问路径**：`/command`

**权限要求**：REVIEWER 或 ADMIN 角色

**展示内容**：
- ✅ **原始数据**（左侧，红色警告风格）
  - 债权人姓名
  - 联系电话
  - 项目名称
  - 城市、区县、地址、房号
  - 面积、工抵价、市场价
  - 上传的凭证文件链接

- ✅ **脱敏数据**（右侧，绿色安全风格）
  - 代号（codeName）
  - 标题（title）
  - 战区标签（locationTag）
  - 安全等级（securityLevel）
  - 类型、面积、容量
  - Token数量、价格
  - 状态（MINTING/AVAILABLE/REJECTED）

**组件**：`CommandCenter.jsx` + `AssetComparisonCard.jsx`

**数据来源**：
- API: `GET /api/arsenal/pending`（需要认证）
- 返回：`{ raw: {...}, sanitized: {...} }`

---

### 2. 首页第四屏（公开）

**访问路径**：`/`（滚动到底部）

**权限要求**：无需登录（公开访问）

**展示内容**：
- ✅ **脱敏资产列表**
  - 资产代号
  - 城市
  - 价格（USDT）
  - 收益率
  - 状态（AVAILABLE/MINTING/RESERVED）
  - 风险等级

**组件**：`AssetsSection.jsx`

**数据来源**：
- API: `GET /api/homepage/assets`（公开）
- 返回：`{ assets: [...] }`（仅脱敏数据）

**展示限制**：
- ❌ 不显示原始数据（敏感信息）
- ✅ 只显示已审核通过（AVAILABLE）或待审核（MINTING）的资产

---

### 3. 资产详情页（公开）

**访问路径**：`/asset-detail/:id`

**权限要求**：无需登录（公开访问）

**展示内容**：
- ✅ **脱敏资产详情**
  - 资产代号、标题
  - 战区信息
  - 安全等级
  - 规格信息
  - 金融信息（Token、价格、收益率）
  - 状态

**组件**：`AssetDetailPage.jsx`

**数据来源**：
- API: `GET /api/arsenal/assets/:id`（公开）
- 返回：格式化的脱敏数据

**展示限制**：
- ❌ 不显示原始数据（敏感信息）
- ✅ 只显示脱敏后的公开数据

---

### 4. 市场页面（公开）

**访问路径**：`/market`

**权限要求**：无需登录（公开访问）

**展示内容**：
- ✅ **脱敏资产列表**（市场格式）
  - 战区（Sector）
  - 类型（Type）
  - 防御等级（Defense）
  - 深度等级（Depth）
  - 容量（Capacity）
  - 价格（Price）
  - 状态（Status）

**组件**：`BlackMarket.jsx`

**数据来源**：
- API: `GET /api/homepage/assets`
- 数据转换：`assetMapper.js` 将脱敏数据转换为市场格式

---

### 5. 地堡页面（用户）

**访问路径**：`/bunker`

**权限要求**：无需登录（支持游客模式）

**展示内容**：
- ✅ **用户资产列表**（如果已登录）
  - 资产代号
  - 城市
  - 状态（已验证/待审核）

**组件**：`BunkerApp.jsx`

**数据来源**：
- API: `GET /api/homepage/assets`
- 根据用户登录状态显示拥有的资产

## 🔍 如何查看数据

### 方法 1: 通过审核台查看（完整数据）

1. 登录具有 REVIEWER 或 ADMIN 角色的账号
2. 访问 `http://localhost:5173/command`
3. 点击左侧待审核资产
4. 查看右侧对比展示（原始数据 + 脱敏数据）

### 方法 2: 通过首页查看（公开数据）

1. 访问 `http://localhost:5173/`
2. 滚动到第四屏（资产列表）
3. 查看已审核通过的资产
4. 点击资产卡片查看详情

### 方法 3: 通过资产详情页查看

1. 访问 `http://localhost:5173/asset-detail/:id`
2. 查看单个资产的详细信息（脱敏数据）

## 💡 数据展示特点

### 1. 权限分离

- **原始数据**：仅审核员可见（审核台）
- **脱敏数据**：所有用户可见（首页、市场等）

### 2. 数据格式

- **审核台**：原始数据 + 脱敏数据对比
- **公开页面**：仅脱敏数据，格式化为用户友好格式

### 3. 实时更新

- 审核批准后，资产状态立即更新
- 首页资产列表自动刷新（30秒）
- 审核台可手动刷新
