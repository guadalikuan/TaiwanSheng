# 图标文件说明

## 所需图标尺寸
- **icon16.png**: 16x16 像素 - 工具栏图标
- **icon48.png**: 48x48 像素 - 扩展管理页面图标  
- **icon128.png**: 128x128 像素 - 安装和商店页面图标

## 图标设计要求
- 主题：TWS (Taiwan Sheng) 相关设计
- 颜色：建议使用蓝色、绿色等积极向上的颜色
- 风格：简洁现代，易于识别
- 建议包含：TWS字母、台湾地图元素、统一相关符号

## 临时解决方案
目前可以使用以下临时图标：
1. 使用简单的彩色圆形作为占位符
2. 在圆形中添加"TWS"文字
3. 使用渐变色背景增强视觉效果

## 图标生成工具推荐
- Figma、Adobe Illustrator 等矢量设计工具
- 在线图标生成器如 Icons8、Flaticon
- AI 图标生成工具如 Midjourney、DALL-E
