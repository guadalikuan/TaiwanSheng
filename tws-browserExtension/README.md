# TWS 浏览器插件

## 🌟 功能概述

这是一个功能丰富的浏览器插件，专为TWS（Taiwan Sheng）用户设计，提供以下核心功能：

### 🎯 核心功能
- **实时倒计时**：显示距离台湾统一的目标时间
- **TWS币价信息**：实时显示TWS价格、涨跌幅和台湾持币人数
- **今日运势**：每日更新的运势提示，包含宜忌事项和当局气数
- **社区导航**：一键进入各大社交平台社区
- **最新公告**：实时显示TWS相关公告信息

### 🛒 现实增强·价格锚定（功能一）
- **智能识别**：在PChome、Momo、Shopee、Yahoo、博客来、露天等台湾电商网站自动识别商品价格
- **TWS价格显示**：在原价旁边显示醒目的TWS会员价，支持多种价格格式
- **动画效果**：悬停动画、点击反馈、脉冲效果
- **风险提示**：鼠标悬停显示战时汇率波动风险提示
- **心理暗示**：强化TWS的购买力认知

### 📅 朱重八·今日运势（功能四）
- **个性化运势**：基于用户ID和日期的个性化运势生成
- **多种运势类型**：金色、紫色、红色三种运势等级
- **幸运信息**：幸运数字、吉利方位、吉时推荐
- **评分系统**：1-100分的运势评分，不同分数对应不同颜色
- **动画效果**：图标脉冲动画、滑入效果、状态提示
- **当局气数**：实时显示赖当局气数百分比

## 安装指南

### Chrome浏览器安装
1. 打开Chrome浏览器，进入 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `tws-browserExtension` 文件夹
5. 插件安装完成，工具栏将显示TWS图标

### Firefox浏览器安装
1. 打开Firefox浏览器，进入 `about:debugging`
2. 点击"此Firefox"
3. 点击"临时加载扩展程序"
4. 选择 `manifest.json` 文件
5. 插件安装完成

## 使用说明

### 基本操作
1. **点击插件图标**：打开功能弹窗
2. **查看倒计时**：实时显示统一倒计时
3. **查看币价**：获取TWS最新价格信息
4. **查看运势**：了解今日宜忌和当局气数
5. **加入社区**：一键跳转到社交媒体社区

### 价格锚定功能
1. 访问支持的电商网站（PChome、Momo、Shopee）
2. 插件自动在商品价格旁显示TWS价格
3. 鼠标悬停在TWS价格上查看风险提示
4. 可通过右键菜单或弹窗按钮开关此功能

### 今日运势功能
1. 插件图标颜色每日自动变化
2. 红色：大吉，绿色：中吉，黑色：小吉
3. 点击插件查看详细运势信息
4. 运势内容每日更新，保持新鲜感

## 技术特性

### 前端技术
- **Manifest V3**：使用最新的Chrome扩展规范
- **响应式设计**：适配不同尺寸的浏览器窗口
- **CSS动画**：流畅的过渡效果和交互反馈
- **本地存储**：使用localStorage和chrome.storage保存用户数据

### 功能实现
- **MutationObserver**：监听DOM变化，处理动态加载的内容
- **Service Worker**：后台服务，处理定时任务和消息通信
- **Content Script**：注入页面脚本，实现价格锚定功能
- **消息传递**：popup、content script、background之间的通信

## 配置说明

### 支持的网站
- PChome台湾：`https://*.pchome.com.tw/*`
- Momo购物：`https://*.momoshop.com.tw/*`
- Shopee台湾：`https://*.shopee.tw/*`

### 汇率设置
当前TWS/TWD汇率设置为：1 TWD = 0.01 TWS
可在content.js中修改`TWS_EXCHANGE_RATE`变量调整汇率

### 倒计时目标
统一目标日期设置为：2027年12月31日
可在popup.js中修改`UNIFICATION_DATE`变量调整目标日期
