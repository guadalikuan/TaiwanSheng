### TWS Protocol 台湾用户使用手册

本手册面向台湾地区的一般用户，帮助你理解并使用 TWS Protocol 提供的各项功能，包括首页四屏、地堡、生存资产市场、我的资产（背包）以及特工系统等模块。文中会尽量采用界面上的中英对照文案，方便你在实际页面中快速找到对应位置。

---

### 一、系统概览与适用场景

- **TWS Protocol 是什么**
  - **形式**: 一个以网页为主的互动式资产与情报可视化系统，整体风格偏军事 / 仿终端界面。
  - **核心元素**:
    - **首页四屏**: 倒计时（OMEGA）、价格与市场（MARKET）、地图雷达（INTEL）、资产卡片（ASSETS）。
    - **功能页面**: 地堡（`BUNKER`）、市场（`MARKET`）、我的资产（`MY LOADOUT / ID`）、特工系统（`AGENT / OPS`）。
    - **地图双视角**: 台湾节点覆盖视图 + 大陆资产池视图。

- **台湾用户的典型使用路径**
  1. 通过网页链接进入首页 `/`，向下滚动查看四个板块。
  2. 如需完整体验（地堡、市场、我的资产、特工系统等），在导航栏中点击 **登录 / 注册**，完成账户开通。
  3. 登录后，从首页或底部导航进入：
     - **地堡 BUNKER**: 查看“生存率”与当前防护状态。
     - **市场 MARKET**: 浏览可用资产卡片、随机分配入口等。
     - **我的资产 MY LOADOUT / ID**: 查看自己名下的资产、收益和可操作按钮。
     - **特工系统 AGENT / OPS**: 查看“身份面板”、网络雷达与任务列表。
  4. 在地图或首页资产卡片中，可进一步进入某个节点/资产的详情页。

- **简化版入口 `/app`**
  - 在部分场景（例如 Telegram 小程序或嵌入式 H5）中，会直接打开简化版页面 `/app`：
    - **UPLINK**: 简化首页视图（倒计时、指数、图表）。
    - **ASSETS**: 简化资产列表。
    - **AGENTS**: 简化特工身份与任务列表。
  - 简化版主要通过底部三按钮在三个视图间切换，而不是使用浏览器路由，适合屏幕较小、希望快速浏览的场景。

### 二、访问方式与推荐环境

- **访问入口**
  - **网页形式**: 运营方通常会提供一个以 `https://…` 开头的网页地址，访问后即进入首页 `/`。
  - **简化版 / 小程序**: 若从 Telegram 等入口进入，多数情况下会直接打开 `/app` 简化界面。

- **推荐设备与网络**
  - **设备**: 建议使用较新的智能手机（iOS / Android）或桌面浏览器（Chrome、Edge、Safari）。
  - **网络**: 需要稳定的网络环境；由于地图与数据为实时更新，如果网络不稳，可能出现加载较慢或动态图卡顿。
  - **浏览器设置**: 确保开启 JavaScript 和本地存储（localStorage），否则登录状态无法保持。

- **隐私与数据**
  - TWS 使用本地浏览器存储 `tws_token` 来维持登录状态，你可以通过“登出 / LOGOUT”清除登录状态。
  - 地图上的台湾节点与大陆资产点，展示的是**脱敏后的聚合信息**，并不会展示具体的个人房产地址。

### 三、账号注册与登录流程

TWS 采用类似 Web3 的“数字身份”体系，账号与链上钱包地址关联。对于台湾用户，可简单理解为“一个用户名 + 一个受助记词保护的账户”。

#### 3.1 注册（IDENTITY REGISTRATION）

入口：
- 首页右上角导航栏点击 **“登录 / LOGIN”** 按钮后，在登录页底部点击 **“立即注册”**。
- 或者直接访问 `/register` 页面。

步骤：

1. **填写基础信息**
   - **用户名 / USERNAME**
     - 要求：3–20 个字符，只能包含字母、数字、下划线。
   - **密码 / PASSWORD**
     - 页面文案提示“至少 6 个字符”，后端实际会要求至少 8 个字符并包含字母和数字。
   - **确认密码 / CONFIRM PASSWORD**
     - 需要与上方密码一致。

2. **配置助记词（MNEMONIC PHRASE）**
   - **模式选择**
     - **自动生成**：系统自动为你创建一段 12 个英文单词的助记词。
     - **手动输入**：你可以填入自己已有的钱包助记词。
   - **自动生成模式**
     - 系统会在页面中展示一串 12 个英文单词，并提示你妥善保存。
     - 可点击“重新生成”重新创建一套助记词。
     - 可点击右侧图标一键复制整段助记词。
     - 页面下方会显示一个对应的钱包地址 `WALLET ADDRESS`，用于提示本次助记词对应的地址。
   - **手动输入模式**
     - 在文本框中输入你已有的 12 个英文单词（以空格分隔）。
     - 系统会校验格式是否合法，并在下方展示推导出的地址。

3. **安全提示**
   - 注册页面会用红色提示强调：
     - 助记词一旦丢失，将无法恢复账户。
     - 不要把助记词公开给他人。

4. **提交注册**
   - 点击 **“注册 / REGISTER”** 按钮，系统会：
     - 在服务器端生成/校验钱包地址。
     - 将密码加密保存，并加密存储助记词。
     - 创建用户档案。
   - 注册成功后：
     - 页面会短暂显示“注册成功，正在跳转到地堡…”
     - 然后自动跳转到 `/bunker` 地堡页面。

#### 3.2 登录（IDENTITY VERIFICATION）

入口：
- 导航栏右侧点击 **“登录 / LOGIN”**，或直接访问 `/login`。

登录方式有两种：

1. **用户名 + 密码登录（默认）**
   - 选择“用户名登录”标签。
   - 填写：
     - **用户名 / USERNAME**
     - **密码 / PASSWORD**
   - 点击“登录 / LOGIN”后：
     - 系统会验证用户名与密码是否匹配。
     - 成功后，将你重定向到之前尝试访问的受保护页面，若无则默认进入 `/bunker`。

2. **助记词 + 密码登录**
   - 切换到“助记符登录”标签。
   - 填写：
     - **助记符 / MNEMONIC PHRASE**：12 个英文单词。
     - **密码 / PASSWORD**：注册时设置的密码。
   - 系统会先校验助记词格式，再通过助记词推导钱包地址，并验证密码是否匹配。

登录状态：
- 登录成功后，系统会在浏览器本地保存一个 `tws_token`，下次打开页面会自动恢复登录。
- 如需退出，可在首页导航栏右上角点击 **“登出 / LOGOUT”** 按钮。

### 四、首页四屏结构与浏览方式

访问首页 `/` 后，页面由上到下分为四个占满屏幕的版块，通过**滚动**或点击导航栏锚点跳转：

1. **天机 | OMEGA（倒计时 + 风险溢价）**
   - 顶部红色警告条，显示系统提醒，例如 `SYSTEM ALERT: GEOPOLITICAL TENSION RISING`。
   - 中央大号数字显示 **“预计时间 ETU (Estimated Time to Unification)”** 倒计时：
     - 格式：`DDDD:HH:MM:SS`。
     - 下方文字标明 `DAYS / HOURS / MINS / SECS`。
   - 下方滚动日志区：
     - 展示最近的“事件触发”文本，如演习、新闻、资金流向等。
   - 底部按钮：
     - **“进入地堡 / ENTER BUNKER”** 按钮，点击后跳转到 `/bunker`。

2. **战图 | MARKET（价格与订单簿概览）**
   - 左侧主区：
     - 顶部显示交易对：`TWS/CNY` 当前价格及 24 小时涨跌幅。
     - 中间为一个简化的 K 线图区域，点击整个图表区域，会跳转到 `/market-detail` 查看更多细节。
   - 右侧侧栏：
     - 展示简化订单簿（买单 / 卖单前几档）。
     - 底部展示最近成交记录（价格 + 时间）。
   - 顶部右侧按钮：
     - **“进入交易”** 按钮，点击后会打开 `/market` 全功能市场页面。

3. **围城 | INTEL（台湾节点 & 大陆资产地图）**
   - 左侧：大陆资产地图
     - 背景为中国大陆区域地图，多个固定的“战略节点”标记（如陕西、福建、广东等）。
     - 顶部信息板展示：
       - 资产池总价值（例如 `¥ 1.425 B`）。
       - “已锁定单位数”以及区块高度。
     - 下方“Recent Asset Confirmations” 列表：
       - 以 `Lot-XXXX` 形式列出最新的资产入库记录和地理归属。
       - 点击其中一条，可跳转到 `/map-asset/:id` 查看单个资产详情。
   - 右侧：台湾节点地图
     - 背景为台湾地区地图，多个红色脉冲光点代表**当前连接节点**。
     - 顶部信息板展示：
       - `Active Nodes` 节点总数。
     - 底部“Live Connect Feed”：
       - 以 `[CONNECT]` 开头的日志条目，代表实时新增或活跃的连接。
       - 点击某条日志，可跳转到 `/map-node/:id` 查看该节点的详细信息。

4. **资产 | ASSETS（资产卡片列表）**
   - 顶部标题：`STRATEGIC RESERVES`，右上方有 **“查看我的资产”** 按钮（跳转 `/loadout`）。
   - 中部网格：多张资产卡片，每张卡片包含：
     - 资产编号：`ID: TWS-xxx`。
     - 状态：`AVAILABLE / RESERVED / LOCKED` 等。
     - 城市信息：标注在卡片预览区域左下角。
     - 入场价格 `Entry Price` 与预期收益率 `Est. Yield`。
     - 底部按钮：如“SECURE TITLE”，表示进行认购/锁定的入口（状态为 LOCKED 时按钮会不可用）。
   - 点击任意卡片，会跳转到 `/asset-detail/:id` 查看详情。
   - 底部脚注区域：
     - 提供“资产入库 / ARSENAL ENTRY”按钮——专供资产供给方使用（录入不动产信息）。
     - 对台湾普通用户来说，这一区域主要是了解“资产从何而来”的流程说明。

### 五、地堡 BUNKER：生存率与基础资产

路径：
- 登录后系统通常会自动将你带到 `/bunker`。
- 也可以从首页的“进入地堡”按钮或底部导航栏 `BUNKER` 进入。

页面结构：

1. **顶部信息栏**
   - 左侧显示一个小圆点（绿/红）代表当前状态是否安全。
   - 显示一个固定 ID，如 `ID: TWS-89757`。
   - 右侧展示 VIP 等级（例如 `VIP: 0`）。

2. **生存率仪表盘（Survival Probability）**
   - 中间为一个环形进度条，显示百分比，如 **34% / 60% / 80%**。
   - 下方有状态标签：
     - `CRITICAL RISK`（高风险）或 `SECURE`（相对安全）。
   - 下方短说明：
     - 当数值较低时，会提示“资产不足以应对事件，需要增加防护资产”之类的文案。

3. **快速行动区（Action Grid）**
   - 两个主要按钮：
     - **SECURE ASSET**：跳转到 `/market` 市场页面，探索或获取新的资产。
     - **OPS MISSION**：本页内部按钮，用于模拟任务操作（提升生存率数值）。

4. **资产金库（ASSET VAULT）**
   - **TWS TOKEN 卡片**：显示当前 TWS Token 余额及状态。
   - **NO SHELTER SECURED 区块**（当你尚未持有资产时）：
     - 红色虚线边框，会显示“尚未拥有庇护所 / 风险等级：极高”等信息。
     - 右侧按钮 **ACQUIRE**，点击后跳转到 `/market`。

5. **底部导航栏**
   - `HOME`：回到首页 `/`。
   - `BUNKER`：当前页。
   - `MARKET`：市场页面 `/market`。
   - `OPS`：特工/任务系统 `/agent`。
   - `ID`：我的资产 `/loadout`。

对台湾用户而言，地堡页可以理解为：
- 你当前整体“防护水平”的可视化面板。
- 通过市场、任务等方式提升防护值，并查看已持有资产是否足够。

### 六、生存资产市场 MARKET

#### 6.1 市场主界面 `/market`

进入方式：
- 从首页 MARKET 区块点击“进入交易”。
- 从地堡或其他页面底部导航点击 `MARKET`。

页面要点：

1. **顶部状态栏**
   - 左上角显示网络状态：`LINK: ENCRYPTED`。
   - 右侧显示服务器时间与时区（如 `UTC+8`）。

2. **盲盒/随机分配模块（RANDOM ALLOCATION）**
   - 标题：`RANDOM ALLOCATION`。
   - 文案描述：系统会根据模拟结果为你分配“最优庇护所”。
   - 按钮：
     - **SECURE NOW 100 USDT**：点击后触发一次模拟分配过程（前端会显示加载及成功提示）。

3. **资产筛选标签**
   - 顶部有 `ALL / NORTH-WEST / SOUTH-WEST / CENTRAL / COASTAL(RISKY)` 等筛选标签。
   - 供你按照区域或属性快速过滤列表。

4. **资产卡片网格**
   - 每张卡片通常包含：
     - `UNIT ID` 及 `SECTOR` 字段（以战区/节点方式描述）。
     - `CLASS / STRAT DEPTH / DEFENSE RATING` 等参数。
     - `REMAINING CAPACITY` 百分比进度条，暗示剩余名额。
     - 右下角显示价格（单位：TWS）。
   - 点击瞄准图标按钮，可进一步操作或关注该资产（具体行为视后续开发而定）。

5. **底部滚动提示**
   - 显示实时提示，例如某区域资产价格变动、某用户刚完成获取等。

6. **底部导航**
   - 与地堡类似，可快速在 `HOME / BUNKER / MARKET / OPS / ID` 间切换。

#### 6.2 市场详情页 `/market-detail`

进入方式：
- 在首页 MARKET 区的 K 线图区域点击。
- 在部分列表中点击订单簿或交易记录入口。

主要内容：

1. **顶部“MARKET DETAIL” 标题 + 返回首页按钮**。
2. **四个 Tab：**
   - `概览`：显示当前价格、24 小时涨跌、交易量、指数等关键指标。
   - `K线图`：展示 60 个历史数据点构成的 K 线图。
   - `订单簿`：详细查看买单/卖单的价格与数量。
   - `交易记录`：按时间列出最近成交记录，标明买入/卖出、价格、数量与时间。

对台湾用户而言，这里类似常见的交易所“行情详情页”，用于观察价格与市场深度，但并不直接等同于真实交易所。

### 七、我的资产 MY LOADOUT / ID

路径：
- 底部导航栏点击 `ID` 进入 `/loadout`。
- 或在首页 ASSETS 区点击“查看我的资产”。

页面结构：

1. **总资产仪表盘（TOTAL NET WORTH）**
   - 显示总资产数值（单位 TWS）及最近 24 小时变动百分比。
   - 下方展示“未领取收益”：
     - `UNCLAIMED RATIONS (YIELD)`，数值单位多为 USDT。
     - 右侧有 **“CLAIM”** 按钮：
       - 点击后会触发一次领取行为，并弹出提示告知领取金额。

2. **资产列表 MY LOADOUT**
   - 标题：`MY LOADOUT (X)`，X 为持有资产数量。
   - 每个资产卡片包含：
     - 名称：如 `TYPE-C SHELTER / UNIT-404`。
     - 位置：如 `SECTOR-CN-NW (Shanxi Node)`。
     - 状态标签：`DEPLOYED` 等。
     - 属性网格：
       - `DEF LEVEL`：等级（LV.1 / LV.3…）。
       - `INTEGRITY`：完整度（%）。
       - `APY`：收益率（%）。
   - 行动按钮：
     - **UPGRADE**：升级防护等级。
     - **BOOST YIELD**：提升收益。
     - 点击后页面会通过提示窗展示扣除的代币数量和操作结果。

3. **紧急清算区 EMERGENCY LIQUIDATION**
   - 红色高亮区域，用于发起卖出/清算：
     - 文案会说明在当前状态下清算会有较高的手续费或惩罚。
     - 按钮：**“INITIATE SELL ORDER (PENALTY APPLIES)”**。
     - 点击会先弹出确认对话框，再执行清算模拟。

整体上，“我的资产”页面给台湾用户提供：
- 当前名下所有资产的聚合视图。
- 每个资产的等级、状态和预估收益。
- 进一步操作（升级、增益、清算）的入口。

### 八、特工系统 AGENT / OPS

路径：
- 底部导航栏点击 `OPS`，进入 `/agent`。

特工系统由三个主要模块组成：特工身份面板（Agent Dashboard）、网络雷达（Network Radar）、任务系统（Mission Log）。

#### 8.1 特工身份面板（Agent Dashboard）

展示内容：
- **CODENAME（代号）**：如 `SHADOW_WOLF`，代表你的特工身份。
- **CURRENT RANK（当前等级）**：如 `STATION CHIEF (站长)`。
- **PROMOTION PROGRESS（晋升进度）**：
  - 百分比条，显示距离下一级还差多少。
  - 下方文字提示下一阶段奖励（例如佣金比例增加等）。
- **赏金统计（Bounty）**：
  - `TOTAL SECURED`：累计获得的赏金金额（单位 USDT）。
  - `PENDING`：待结算金额，通常在某些条件达成后才能释放。

邀请码区域：
- 标题：`RECRUITMENT FREQUENCY`。
- 主要元素：
  - `SECURE LINK`：显示你的专属邀请码/代码（例如 `TWS-8821-X`）。
  - 点击该区域，会将邀请语 + 链接复制到剪贴板，方便分享到外部社交渠道。

#### 8.2 网络雷达（Network Radar）

视觉效果：
- 中心白点代表你自己，周围绿色/灰色小点代表不同的“节点”或“联系人”。
- 外圈的同心圆与旋转扫描线，营造“雷达扫描”效果。

数据区：
- 顶部显示：
  - `ACTIVE CELLS`：活跃节点数量。
  - `SLEEPERS`：沉睡/不活跃节点数量。
  - `DEPTH`：网络深度等级（如 `LVL 4`）。
- 底部“LIVE FEED”：
  - 滚动展示最新增加的节点日志，如 `NEW CELL DETECTED…`。

对台湾用户的意义：
- 你可以将其理解为“邀请网络 / 联系人网络”的可视化展示，用来观察自己扩展网络的情况。

#### 8.3 任务系统（Mission Log）

任务列表：
- 标题：`DAILY PROTOCOLS`。
- 上方有一个系统状态条：如 `DEFCON 3`。
- 列表中每个任务包含：
  - 标题（如 `ESTABLISH UPLINK` / `DEPLOY PSY-OPS`）。
  - 简要说明。
  - 奖励数额（多少 TWS）。
  - 当前状态：
    - `pending`（待完成）/ `completed`（已完成）/ `locked`（未解锁）。

操作方式：
- 对于状态为 `pending` 的任务，会有一个操作按钮（例如 `CONNECT / DEPLOY / UPLOAD`）。
- 点击后：
  - 控制台日志区会输出执行过程。
  - 稍后任务状态会变为 `completed`，并可能解锁下一项任务。

对台湾用户而言，该模块可以看作是：
- 指导你“每天可以做些什么”的任务列表。
- 带有一定故事性和仪式感的签到/分享/上传动作集合。

### 九、地图与台湾用户可见的“脱敏资产数据”

系统在地图和资产列表中，向台湾用户展示的主要是**脱敏后的聚合数据**，而不是具体的个人隐私信息。

#### 9.1 台湾视角：节点覆盖网络

在首页 `INTEL` 区域或独立 `tw.html` 页面中：
- 你会看到：
  - 台湾地区地图上不断出现的脉冲光点。
  - `Active Nodes` 数字不断增长。
  - `Live Connect Feed` 日志中出现不同地区的连接记录（如 `Taipei (Xinyi)` 等）。
- 点击地图上的节点（在应用中）时：
  - 会跳转到 `/map-node/:id`。
  - 在 **节点详情页** 中，你可以看到：
    - 节点代号（ID）。
    - 所属城市与大致位置。
    - 最近连接时间。
    - 与该节点相关的历史日志列表。

这些信息都是基于日志与节点编号的抽象，不会暴露具体用户的真实姓名或精确住址。

#### 9.2 大陆视角：资产避难所网络

在首页 `INTEL` 区域左侧或独立 `mainland.html` 页面中：
- 你会看到：
  - 中国大陆地图上若干“金色信标”，代表主要资产节点（如陕西、福建等）。
  - 顶部显示 **资产池总价值** 与 **已锁定单位数**。
  - 底部“Recent Asset Confirmations” 列表展示：
    - `Lot-XXXX` 编号。
    - 所属区域描述（如 `SHAANXI / FUJIAN` 等）。
    - 确认可视为资产已入池的记录。

点击这些列表或地图上的闪烁点时：
- 会跳转到 `/map-asset/:id`。
- 在该页面你可以看到：
  - 资产编号 `Lot-XXXX`。
  - 大致位置（城市 / 区域）。
  - 价值数额。
  - 所属节点及坐标。
  - 整个资产池的汇总统计（总价值、总单位数等）。

对于台湾用户来说，这些都是从资产原始信息中**脱敏、聚合后**得出的可视化数据。

### 十、常见问题与操作提示

- **1. 忘记密码怎么办？**
  - 如果你还保留了正确的 12 个单词助记词，可以通过“助记符登录”重新登录，再在设置中更改密码。
  - 如果同时丢失了密码和助记词，则无法找回原账户，只能重新注册。

- **2. 登录后页面是英文/术语较多，看不懂怎么办？**
  - 页面大部分区域都有中英文标签，例如“地堡 / BUNKER”、“身份验证 / IDENTITY VERIFICATION”等。
  - 本手册各章节按照实际页面结构编排，可以对照查看。

- **3. 地图和数据经常在动，是正常的吗？**
  - 是正常现象。地图节点、资产池数值、订单簿和交易记录都设计为定期刷新，表现为“实时流动”的效果。
  - 如果长时间未更新，建议检查网络或刷新页面。

- **4. 如何退出登录？**
  - 在首页顶部导航栏右上角找到用户信息区域，点击 **“登出 / LOGOUT”**。
  - 退出后，浏览器中保存的 `tws_token` 会被清除，下次访问会回到未登录状态。

- **5. 台湾用户是否需要懂区块链才能使用？**
  - 不需要深入的技术背景。
  - 你只需要记住：
    - 助记词 = 你账户的“总钥匙”。
    - TWS 会自动处理幕后与区块链相关的技术细节。

### 十一、总结

对于台湾用户来说，TWS 提供的是一个高度可视化的“情景式资产与网络面板”。  
你可以从首页快速理解整体环境（倒计时、价格、地图）、通过地堡了解自己的“防护状态”，在市场与资产页面查看和管理自己的资产，并通过特工系统中的任务与网络雷达维持一定互动频率。  
在整个过程中，请务必妥善保管你的助记词，并在任何涉及资产变动的操作前，充分理解界面上的提示与说明。
