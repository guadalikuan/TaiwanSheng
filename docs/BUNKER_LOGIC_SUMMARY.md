# 地堡逻辑简述

## 🎯 核心逻辑

**目的**：避险（应对"统一事件"的恐惧）  
**手段**：买币（TWS代币）+ 买房子（大陆房产）

---

## 📊 生存率计算公式

```
生存率 = 34% + 代币避险加成 + 房产避险加成 + 组合加成 - 风险惩罚
```

---

## 💰 避险手段详解

### 1. 买币（TWS代币）

**机制**：
- 每 10,000 TWS = +1% 生存率
- 最高加成：30%
- 作用：流动性避险，可以随时变现

**示例**：
- 50,000 TWS = +5%
- 300,000 TWS = +30%（上限）

---

### 2. 买房子（大陆房产）

**机制**：
```
每个房产加成 = 15% + (位置系数 × 5%) + (面积系数 × 3%)
```

**位置系数**（覆盖全国400+城市）：
- 新疆、西藏：2.0（+10%）
- 西北、西南：1.5-1.8（+7.5%-9%）
- 华中、华北：1.0-1.5（+5%-7.5%）
- 华东、华南：0.5-1.0（+2.5%-5%）

**面积系数**：
- 单兵舱（<50㎡）：0.5（+1.5%）
- 避难所（50-90㎡）：1.0（+3%）
- 地堡（90-140㎡）：1.5（+4.5%）
- 指挥所（>140㎡）：2.0（+6%）

**示例**：
- 乌鲁木齐地堡（120㎡）= 15% + (2.0 × 5%) + (1.5 × 3%) = **27%**
- 上海避难所（80㎡）= 15% + (0.7 × 5%) + (1.0 × 3%) = **21.5%**

---

### 3. 组合加成

**机制**：
- 同时持有代币和房产，总加成提升 10%
- 公式：`(代币加成 + 房产加成) × 10%`

**示例**：
- 50,000 TWS（+5%）+ 1个乌鲁木齐地堡（+27%）
- 组合加成 = (5% + 27%) × 10% = 3.2%
- 总加成 = 5% + 27% + 3.2% = **35.2%**

---

### 4. 风险惩罚

**机制**：
- 地缘政治风险上升时，生存率下降
- CRITICAL：-20%
- HIGH：-10%
- MEDIUM：0%
- LOW：+5%（奖励）

---

## 🔄 用户行为路径

```
进入地堡（34% 极度危险）
    ↓
购买代币（+5% → 39%）
    ↓
购买房产（+27% → 66% 安全）
    ↓
风险上升（-20% → 46% 危险）
    ↓
继续购买（达到 100% 完全安全）
```

---

## 💡 心理作用

1. **量化安全感**：将抽象的"安全"转化为具体数字（34% → 100%）
2. **控制感**：用户通过购买资产，感觉自己"控制"了生存率
3. **紧迫感**：风险上升时，生存率下降，驱动用户购买更多资产
4. **真实感**：动态调整让用户感觉"这是真的，不是游戏"

---

## ✅ 总结

**一句话**：
> 地堡通过"买币+买房子"提升生存率，从34%（极度危险）到100%（完全安全），实现避险目的。

**核心公式**：
```
生存率 = 34% + 代币避险 + 房产避险 + 组合加成 - 风险惩罚
```

**关键点**：
- 代币：流动性避险（每10,000 TWS = +1%，最高30%）
- 房产：物理避险（每个房产 +15% 起，位置和面积影响加成）
- 组合：协同效应（同时持有，额外+10%）
- 风险：动态调整（风险上升，生存率下降）
